# Accessories-Hub-E-commerce-Website
Accessories Hub is a web-based e-commerce system for buying electronic accessories like earbuds and headphones. It includes user registration, login, product browsing, cart, and order features, along with an admin panel for managing products, orders, and users, built using HTML, CSS, JavaScript, PHP, and MySQL.


Accessories Hub

Accessories Hub is a web-based e-commerce application designed for purchasing electronic accessories such as earbuds, headphones, and related gadgets. The project provides a smooth and user-friendly interface for customers while offering an efficient admin panel for management.

Features

User Side
User registration and login system
Browse available products
Add products to cart
Place and manage orders
Admin Side
Add, update, and delete products
Manage customer orders
Monitor users and activity
Technologies Used
Frontend: HTML, CSS, JavaScript
Backend: PHP
Database: MySQL

Purpose
This project was developed to demonstrate full-stack web development skills, including frontend design, backend logic, and database integration in a real-world e-commerce scenario.

How to Run

Clone the repository
Import the database into MySQL
Configure database connection in PHP files
Run the project using a local server (e.g., XAMPP)
Open in browser: http://localhost/your-project-folder
